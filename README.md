# BogoDb

BogoDb is a toy database management system written in Go.
Inspired by CMU Database Group's Lecture (of course not homework!)

## Specification 

- SQL
    - create table
    - insert statment
    - select statement(from, where)
    - update statment
- Index(with b-tree)
- Buffer on memory
- Concurrency(only transaction)
- not mmap implementation
